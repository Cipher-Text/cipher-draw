# Contributing

## Setup
1. Install Node.js 20+ and pnpm.
2. Run `pnpm install`.
3. Start development with `pnpm -C apps/web dev`.

## Standards
- TypeScript strict mode is enabled.
- Run `pnpm lint` before opening PRs.
- Run `pnpm test` for unit tests.

## Pull Requests
- Keep PRs scoped and include a clear summary.
- Add tests for bug fixes and new logic where possible.
